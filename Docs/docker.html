    • Using XData on docker

    1. Install Docker :
      • Follow the installation steps shown here
      • Verify docker has correctly been intalled using : $ sudo docker run hello-world
      • Also do the following post-installation steps for user priviledges (as given in the website) :
        $ sudo groupadd docker
        $ sudo usermod -aG docker $USER
        [Log out and log back in so that your group membership is re-evaluated. If testing on a virtual machine, it may be necessary to restart the virtual machine for changes to take affect. On a desktop Linux environment such as X Windows, log out of your session completely and then log back in.]
    2. Unzip the 'Xdata' directory
    3. Open a terminal in the Xdata directory
    4. $ docker build -t xdata:1 .
      [This command builds a docker image from dockerfile. Note the dot (.) at the end of the command. This command may take some time.]
    5. $ docker run -it -p ‹port_no›:8080 xdata:1
      [This command runs the docker image. Use any free port number in the ‹port_no› slot. ]
    6. Open the link: 'localhost:‹port_no›/XDataGrading/' in your local browser. If you are running docker on a different machine, replace localhost with the network address of the mahcine.
      [Use the same free port number as step 5 ]
    7. Use 'exit' command in terminal to exit from docker.
      [Steps 2-5 has to be done only once for the initial creation of docker image and running it. Subsequently only the following steps are sufficient for running dockerized Xdata]
    8. Open terminal and run $ docker ps -a
    9. Note the container-id of the latest xdata:1 image
    10. $ docker start ‹container-id›
      [Use the container-id obtained from above for the following command]
    11. $ docker attach ‹container-id› [And press enter]
    12. Wait for a few seconds and follow steps 6 & 7
