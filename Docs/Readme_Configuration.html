    • Developer Setup

    This specifies the system setup required for developer.

    1. Install the CVC3 and Apache tomcat server.
    2. Install PostgreSQL (version > 9.1).
    3. Create the database user specified in the installer file. To use the default settings create user 'testing1' with password 'testing1' and skip steps 4 and 5 .
    4. Modify the Install.sh file to change the database settings for the required database.
    5. Modify the file XData.properties file in xdata/util to change the database settings
    6. Run the Install.sh as a sudo user
    7. Install Eclipse EE Edition (Luna) and JDK 1.6/1.7. Set up the appropriate environment variables and try running SimpleTest.java 
       
public class SimpleTest {
    public static void main(String[] args) {
	System.out.println("This works!");
    }
}
    8. Click on File->Import and under Java->Existing Projects into workspace. Browse to the root of XData directory and click Add.
    9. Go to Window->Show View->Others and select Servers. Click on add a new server. Select Tomcat 7.0 and set it up. In the end, add XData to the Tomcat Server projects and close the window.
    10. In tool.jsp set the debug flag to 1, i.e. debug = 1, to run the project without the need of integrating with Moodle.
    11. Build your project, correct the build errors if any, start the server and open the following URL in the browser 
      localhost:8080/XDataWeb/index.html
