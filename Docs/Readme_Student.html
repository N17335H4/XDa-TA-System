    • Readme for Students

      Login instructions
    1. Learning Management System login: The assignment link on the course page will take you to the assignment.
    2. Direct login: The URL provided will take you to a login screen. Enter you login credentials to go the course list page. In the course list page select the course <couse-id> to get the list of assignments for the course. Then select <assignment-name> to go to today's assignment where you can see the list of questions along with the submission deadline. The assignment questions are available only after the Start Time. You will not be allowed to make submissions once the deadline is over.
      Graded assignment instructions
    1. You can click the Edit link with each question to add/edit you answer. You need to click submit after writing your answer. This will save your answer, evaluate it against one basic test case and will let you know if the query is syntactically correct and if it is correct on the test case.
    2. You may submit your answer multiple times. Your last submission will be counted towards your grade.
    3. After the assignment is graded you can see the list of your answers along with the marks awarded to you
    4. If any answer is incorrect you can see the test cases on which your answer fails along with the type of error that the test case was meant to catch. You can also see the weightage given to partial marking parameters and how much marks is awarded to you
      Learning mode instructions
    1. You can click the Edit link with each question to add/edit you answer. You need to click submit after writing your answer. This will save your answer, evaluate it against the answers of the instructor and will let you know if the query is correct.
    2. You can submit your answer multiple times.
