    • Instructor Usage

      Partial Marking
    1. The partial marking feature in the grading tool assigns marks to student queries based on a heuristic of how close the student is to to the instructor's query. It ensures to ensure a standard and consistent evaluation scheme across all students.
    2. Partial marking is done only if the student query fails one or more datasets during evaluation. For these student submissions the instructor can click on the Details link in Question Status page to see how the partial marks were awarded.
    3. The instructor can use the View/Adjust Partial Marking parameters button to see details of how the partial marking was done. It shows the parameters that were used for grading and the marks allotted for each parameter. The parameters currently include the relations used, projected attributes and query conditions.

      4. The instructor can adjust the parameters to change the marks. The instructor can also override the marks and award marks as seen fit by him.
